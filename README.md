# Remaining Life – Life in Months Visualization

This project is a simple visualization that shows **how much of your life has already passed and how much remains**, represented in **months**.

The idea is inspired by the concept of visualizing life in small units of time to make us more aware of how we spend it. Each **circle represents one month of life**.

When you enter your **birth date** and an **expected lifespan**, the application calculates how many months you have lived and visually displays them.

---

## How it Works

The visualization is built using circles:

- 🔴 **Red circles** → Months already lived  
- ⚪ **White circles** → Months remaining  

---

## Features

- Visual representation of **life in months**
- Dynamic update based on **birth date input**
- Adjustable **expected lifetime**
- Simple and minimal interface
- Responsive layout

---

## Tech Stack

This project is built using:

- **Next.js**
- **React**
- **TypeScript**
- **Tailwind CSS**
- **shadcn/ui**

## Inspiration

This idea was inspired by the concept of visualizing life as a limited number of time units.

Seeing life in small measurable pieces makes you realize **how valuable time really is**.

## Running the Project

Clone the repository:
```bash
git clone <repo-url>
```

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Open the app at:
```bash
http://localhost:3000
```
